# Time in a Bottle

Conversor de tempo de RPG onde o tempo passa de forma diferente da vida real.

## Sobre

Time in a Bottle é uma calculadora que converte tempo real em tempo de jogo para RPGs que utilizam um sistema de passagem de tempo acelerado.

## Sistema de Tempo

O sistema funciona com as seguintes regras:

- 1 semana real = 1 mês no RPG (30 dias)
- 1 mês real = 1 ano no RPG (120 dias)
- 1 ano no RPG possui apenas 4 meses
- O tempo passa aproximadamente 4.3x mais rápido no jogo

## Recursos

- Conversão de horas reais para tempo de RPG
- Exibição em formato legível (anos, meses, semanas e dias)
- Cálculo automático de experiência acumulada
- Interface responsiva para mobile
- Design baseado nas cores do Discord

## Sistema de XP

A cada 3 semanas de jogo, o personagem ganha:
- 5.000 pontos de experiência
- Conclusão de atividades em andamento

## Tecnologias

- HTML5
- CSS3
- JavaScript Vanilla

## Como Usar

1. Digite o número de horas que passaram na vida real
2. O sistema calculará automaticamente quanto tempo passou no RPG
3. Visualize o tempo em formato detalhado e o XP acumulado
